## Installation

This package can be installed
by adding `dsm_parser` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:dsm_parser, "~> 0.1.1"}
  ]
end
```

Documentation (when it is written) can be found at [https://hexdocs.pm/dsm_parser](https://hexdocs.pm/dsm_parser).
